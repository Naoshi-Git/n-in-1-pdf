# N-in-1 PDF Layout Tool 📄

高性能でプライバシーに配慮した、ブラウザ完結型のPDF分割・結合レイアウトツールです。

## 🚀 特徴
- **完全ローカル処理**: PDFデータはブラウザ内だけで処理され、サーバーに送信されることはありません。
- **直感的な操作**: ドラッグ＆ドロップで読み込み、プレビューを見ながら拡大縮小、除外、空白挿入が可能。
- **高品質出力**: ベクターデータを保持したまま、高品質なPDFを生成。
- **Google AI Studio風 UI**: モダンで洗練されたダークモードインターフェース。

## 📖 使い方
1. 公開URLにアクセスします。
2. PDFファイルをドラッグ＆ドロップします。
3. N-in-1設定やズームを調整します。
4. 「PDFとして書き出し」をクリックして保存！

## 🛠 技術スタック
- HTML / Vanilla CSS / JavaScript
- [pdf.js](https://mozilla.github.io/pdf.js/) (PDF rendering)
- [pdf-lib](https://pdf-lib.js.org/) (PDF manipulation)
